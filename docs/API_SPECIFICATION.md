Документация API
Эндпоинты


GET /api/dashboard/metrics

Описание: Получение текущих метрик.
Ответ:
json
Copy

{
  "pensioners_count": 0,
  "volunteers_count": 0,
  "last_updated": "2023-01-01T00:00:00",
  "status": "active"
}




GET /api/dashboard/history

Описание: Получение исторических данных.
Параметры:

period (строка): Период для отображения (7d, 30d, 90d, all).

Ответ:
json
Copy

{
  "period": "30d",
  "pensioners": [
    {"date": "2023-01-01", "count": 10},
    {"date": "2023-01-02", "count": 15}
  ],
  "volunteers": [
    {"date": "2023-01-01", "count": 5},
    {"date": "2023-01-02", "count": 10}
  ]
}



Обработка ошибок
Приложение обрабатывает следующие ошибки:

Ошибки аутентификации: Если не удается получить токен для доступа к API.
Ошибки подключения: Если не удается подключиться к API.
Ошибки обработки данных: Если данные не могут быть обработаны.
Все ошибки логируются и возвращаются клиенту в виде сообщений об ошибках.
